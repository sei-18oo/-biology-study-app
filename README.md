# 生物基礎 学習アプリ（PWA版）

## 📱 特徴

- **PWA対応**: スマホにインストール可能
- **オフライン対応**: ネット接続なしでも使用可能
- **全6大問・105問**: 定期試験の全範囲をカバー
- **AI分析機能**: 学習パターンを詳細に分析
- **ヒント・解説付き**: 各問題に小ヒントと大ヒント

## 📚 学習内容

### 大問1：細胞の構造（10問）
- 真核細胞と原核細胞
- 細胞小器官
- 単細胞生物と多細胞生物

### 大問2：酵素・呼吸・光合成（10問）
- 酵素の性質
- 呼吸と光合成の化学式

### 大問3：DNA・遺伝・タンパク質合成（20問）
- DNAの塩基対
- 転写と翻訳
- ゲノムと相同染色体

### 大問4：体内調節（20問）
- 自律神経系
- ホルモン分泌
- 血糖濃度調節

### 大問5：免疫（20問）
- 自然免疫と獲得免疫
- 抗原と抗体
- 免疫疾患

### 大問6：論述問題（5問）
- 50字以上の記述問題

## 🚀 使い方

### スマホにインストール

1. ブラウザでアプリを開く
2. 画面右下の「📱 アプリをインストール」ボタンをタップ
3. 「ホーム画面に追加」を選択

### 学習の進め方

1. メニューから学習したい大問を選択
2. 問題に解答
3. ヒントが必要な場合は「小ヒント」ボタン
4. 詳しい解説は「大ヒント」ボタン
5. 解答を送信して正誤を確認
6. 全問終了後、AI分析を確認

## 💡 機能

- **学習行動追跡**: 解答時間、迷った回数、ヒント使用状況を記録
- **弱点分析**: 時間がかかった問題、迷った問題を自動判定
- **AI分析**: 個別最適化されたアドバイスを提供
- **全体分析**: 得意分野・苦手分野をグラフで可視化

## 🔧 技術仕様

- React 18
- Tailwind CSS
- Recharts（グラフ表示）
- Service Worker（オフライン対応）
- PWA（Progressive Web App）
